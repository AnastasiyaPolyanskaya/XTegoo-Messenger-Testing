# 3 - XTegoo Messenger Favorites & Saved Messages Testing

Практический проект по ручному тестированию мобильного приложения XTegoo Messenger.

В рамках проекта проведено тестирование функционала «Избранное», связанного с сохранением сообщений и доступом к ним через различные разделы приложения.

---

## Цель проекта

Проверить корректность работы функции сохранения сообщений в избранное, а также доступность и отображение сохранённых данных в интерфейсе приложения.

---

## Объект тестирования

Мобильное приложение XTegoo Messenger

- Платформа: iOS  
- Тип тестирования: Manual QA  
- Устройство: iPhone 14 Pro Max  

---

## Области тестирования

### 1. Сохранение сообщений

Проверено:
- добавление сообщений в избранное;
- корректность выполнения действия «добавить в избранное»;
- сохранение данных после выполнения действия.

---

### 2. Отображение избранного

Проверено:
- отображение сохранённых сообщений в разделе «Избранное»;
- доступ к избранному через профиль пользователя;
- соответствие фактического результата ожидаемому.

---

### 3. Доступ к избранному через чаты

Проверено:
- отображение вкладки «Избранное» в списке чатов;
- доступность функции через основной экран приложения;
- корректность навигации.

---

## Особенности тестирования

Тестирование проводилось:
- с использованием реальных пользовательских сценариев;
- с проверкой отображения данных в разных разделах приложения;
- с акцентом на пользовательский опыт и доступность функций.
