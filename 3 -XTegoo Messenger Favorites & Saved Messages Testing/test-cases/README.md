# Test Cases — XTegoo Messenger (Favorites & Saved Messages)

В данном разделе представлены тест-кейсы для проверки функциональности «Избранное» в мобильном приложении XTegoo Messenger.

Тест-кейсы охватывают пользовательские сценарии, связанные с сохранением сообщений и доступом к ним через различные разделы приложения.

---

## Цель тестирования

Проверить корректность работы функции сохранения сообщений в избранное, а также доступность и отображение сохранённых данных.

---

## Область покрытия

Тест-кейсы охватывают следующие функциональные области:

### 1. Сохранение сообщений
- добавление сообщений в избранное;
- корректность выполнения действия «добавить в избранное»;
- сохранение данных после добавления.

---

### 2. Отображение избранного

- отображение сохранённых сообщений в разделе «Избранное»;
- доступ к избранному через профиль пользователя;
- соответствие отображаемых данных ожидаемому результату.

---

### 3. Доступ через чаты

- отображение вкладки «Избранное» в списке чатов;
- доступность функции через основной экран приложения;
- корректность переходов между экранами.

---

## Структура тест-кейсов

Каждый тест-кейс содержит:

- ID тест-кейса  
- Название  
- Предусловия  
- Шаги выполнения  
- Ожидаемый результат (ОР)  
- Фактический результат (ФР)  
- Окружение  
- Результат выполнения  

---

## Окружение

Тестирование проводилось на:

- iPhone 14 Pro Max  
- iOS 26.6.4  

---

## Результаты выполнения

В ходе выполнения тест-кейсов:

- часть сценариев завершилась со статусом Passed  
- часть сценариев завершилась со статусом Failed  

Выявленные проблемы:

- сообщения не отображаются в разделе «Избранное»;
- вкладка «Избранное» отсутствует в чатах;
- функционал работает некорректно.

---

## Примечание

Тест-кейсы оформлены в табличном виде и отражают реальные пользовательские сценарии взаимодействия с приложением.
