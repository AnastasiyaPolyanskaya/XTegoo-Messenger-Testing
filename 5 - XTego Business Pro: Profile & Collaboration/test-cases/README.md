Назначение документа

Данный документ содержит набор тест‑кейсов для проверки:

    корректности отображения данных бизнес‑аккаунта;

    функциональности обмена бизнес‑контактами;

    работы справочных разделов приложения;

    синхронизации данных между устройствами;

    чат‑функционала (быстрые ответы, кнопка «Написать компании»).

Каталог тест‑кейсов
TC‑01: Проверка визуального отображения адреса сайта

Приоритет: Высокий (High)
Тип: Функциональный
Предусловие:

    Пользователь зарегистрирован в системе.

    Создан бизнес‑аккаунт.

    Добавлен каталог товаров (тюльпаны).

    Заполнен адрес сайта в настройках.

Шаги выполнения:

    Открыть настройки приложения.

    Нажать на раздел «Ссылки».

    Нажать на пункт «Сайт».

Ожидаемый результат: полное отображение адреса сайта без обрезания текста.
Фактический результат: адрес сайта отображается наполовину, часть текста обрезана.
Статус: Failed
Окружение: iOS 16.5, XTegoo Messenger v2.3.1

Выводы и рекомендации

Ключевые проблемы:

    некорректное отображение данных (адрес сайта, бизнес‑контакты);

    неработающие кнопки в разделе настроек;

    проблемы с синхронизацией между устройствами;

    ошибки в работе чат‑функционала.

Рекомендации по исправлению:

    Исправить обрезку текста при отображении адреса сайта.

    Реализовать полное отображение бизнес‑контактов в чатах (имя + телефон + email).

    Активировать кнопки справочных разделов («Помощь», «Вопросы о XTego», «Возможности XTego»).

    Наладить синхронизацию данных между устройствами.

    Устранить проблему с отправкой быстрых ответов в чатах.

Следующие шаги:

    передать баг‑репорты разработчикам;

    запланировать ретест после исправления дефектов;

    расширить покрытие тест‑кейсами для смежных функций.
