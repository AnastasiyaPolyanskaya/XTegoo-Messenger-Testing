Тип тестирования: ручное (Manual QA)
Назначение

Документ содержит набор тест‑кейсов для проверки:

    настроек бизнес‑аккаунта;

    корректности отображения данных;

    работы функций обмена бизнес‑контактами;

TC‑01. Проверка визуального отображения адреса сайта

Предусловие:

    Пользователь зарегистрирован.

    Создан бизнес‑аккаунт.

    Добавлен каталог товаров (тюльпаны).

    Заполнен адрес сайта.

Шаги выполнения:

    Открыть настройки приложения.

    Нажать «Ссылки».

    Нажать «Сайт».

Фактический результат: название сайта отображается наполовину.

Приоритет: High
