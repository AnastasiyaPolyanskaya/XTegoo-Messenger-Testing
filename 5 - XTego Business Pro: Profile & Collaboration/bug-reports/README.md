BUG‑01: Обрезание адреса сайта при отображении

Заголовок: Обрезается адрес сайта в настройках приложения
Приоритет: Высокий (High)
Серьёзность: Критическая (Critical) — нарушает базовую функциональность профиля
Предусловие:

    Создан бизнес‑аккаунт.

    Заполнен адрес сайта.

Шаги воспроизведения:

    Нажать «Ссылки».

Фактический результат: адрес отображается наполовину, часть текста не видна.

BUG‑02: Неполное отображение бизнес‑контакта в чатах

Заголовок: В чатах отображается только имя бизнес‑контакта, отсутствуют телефон и почта
Приоритет: Высокий (High)
Серьёзность: Высокая (High) — снижает полезность функции обмена контактами
Предусловие:

    Пользователь зарегистрирован.

    Создан бизнес‑аккаунт.

    Добавлен каталог товаров (тюльпаны).

    Заполнены контактные данные (телефон, почта).

Шаги воспроизведения:

    Открыть настройки приложения.

    Перейти в «Мой профиль».

    Нажать «Поделиться бизнес‑контактом».

    Выбрать контакт «Ленар Сафин».

    Открыть чат с Ленаром Сафиным.

Ожидаемый результат: в чате отображается полный бизнес‑контакт: имя, кликабельный номер телефона, кликабельная почта.
Фактический результат: в чате видно только имя, остальные данные отсутствуют.
Окружение: iOS 16.5, XTegoo Messenger v2.3.1
Воспроизводимость: 100 %
Вложения: screenshots/bug_02_contact_incomplete.png
Рекомендации по исправлению: проверить передачу контактных данных при отправке, убедиться, что все поля передаются и отображаются корректно.
