Назначение документа

Документ содержит описание скриншотов, сделанных в ходе тестирования бизнес‑функционала приложения XTegoo Messenger. Каждый скриншот:

    фиксирует дефект или корректную работу интерфейса;

    сопровождается описанием проблемы;

    помогает разработчикам быстро локализовать и исправить ошибки.

Каталог скриншотов
SC‑01: Обрезание адреса сайта в настройках

Описание: скриншот демонстрирует проблему с отображением адреса сайта — текст обрезан, часть информации не видна.
ID баг‑репорта: BUG‑01
Шаги воспроизведения:

    Открыть настройки приложения.

    Нажать «Ссылки».

    Нажать «Сайт».
    Что выделено на скриншоте: область с обрезанным текстом (красная рамка).
    Файл: screenshots/bug_01_address_truncated.png
    Дата съёмки: 30.04.2026
    Окружение: iOS 16.5, XTegoo Messenger v2.3.1

SC‑02: Неполное отображение бизнес‑контакта в чате

Описание: в чате отображается только имя бизнес‑контакта («Ленар Сафин»), отсутствуют номер телефона и почта.
ID баг‑репорта: BUG‑02
Шаги воспроизведения:

    Открыть настройки приложения.

    Перейти в «Мой профиль».

    Нажать «Поделиться бизнес‑контактом».

    Выбрать контакт «Ленар Сафин».

    Открыть чат с Ленаром Сафиным.
    Что выделено на скриншоте: строка с именем контакта (жёлтая стрелка).
    Файл: screenshots/bug_02_contact_incomplete.png
    Дата съёмки: 30.04.2026
    Окружение: iOS 16.5, XTegoo Messenger v2.3.1

SC‑03: Неактивная кнопка «Помощь» в настройках

Описание: кнопка «Помощь» визуально неактивна (серая), клик не вызывает реакции.
ID баг‑репорта: BUG‑03
Шаги воспроизведения:

    С главной страницы нажать «Настройки».

    Внутри настроек нажать ещё раз «Настройки».

    Найти кнопку «Помощь».
    Что выделено на скриншоте: неактивная кнопка (красная рамка + значок «X»).
    Файл: screenshots/bug_03_help_button_inactive.png
    Дата съёмки: 30.04.2026
    Окружение: iOS 16.5, XTegoo Messenger v2.3.1

SC‑06: Ошибка синхронизации «Просмотра записей» между устройствами

Описание: на втором устройстве при открытии «Просмотра записей» отображается ошибка или пустой экран.
ID баг‑репорта: BUG‑06
Предусловие:

    Устройство 1: добавлен каталог товаров.

    Устройство 2: попытка открыть записи.
    Шаги воспроизведения:

    Устройство 2: открыть настройки приложения.

    Устройство 2: перейти в раздел «Просмотр записей».
    Что выделено на скриншоте: пустой экран/сообщение об ошибке (красная рамка).
    Файл: screenshots/bug_06_sync_error.png
    Дата съёмки: 30.04.2026
    Окружение:

    Устройство 1: iOS 16.5, XTegoo v2.3.1;

    Устройство 2: iOS 16.5, XTegoo v2.3.1.

SC‑07: Не отправляется быстрый ответ в чате

Описание: после выбора шаблона быстрого ответа («Здравствуйте, чем могу помочь?») сообщение не появляется в чате.
ID баг‑репорта: BUG‑07
Шаги воспроизведения:

    Открыть диалог с Ленаром Сафаровым.

    Нажать на иконку скрепки.

    Выбрать «Быстрые ответы».

    Нажать на шаблон «Здравствуйте, чем могу помочь?».
    Что выделено на скриншоте: чат без отправленного сообщения (жёлтая стрелка на пустое поле ввода).
    Файл: screenshots/bug_07_quick_reply_fail.png
    Дата съёмки: 30.04.2026
    Окружение: iOS 16.5, XTegoo Messenger v2.3.1
