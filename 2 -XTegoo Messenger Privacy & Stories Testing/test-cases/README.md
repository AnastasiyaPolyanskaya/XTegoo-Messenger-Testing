# Test Cases - XTegoo Messenger (Privacy & Stories)

В данном разделе представлены тест-кейсы для проверки функциональности сторис и настроек приватности в мобильном приложении XTegoo Messenger.

Тест-кейсы охватывают ключевые пользовательские сценарии, связанные с созданием сторис, управлением доступом и отображением контента.

---

## Цель тестирования

Проверить корректность работы функционала сторис и настроек приватности, а также выявить дефекты, влияющие на доступ к контенту и пользовательский опыт.

---

## Область покрытия

Тест-кейсы охватывают следующие функциональные области:

### 1. Создание сторис
- добавление фото в сторис;
- публикация сторис;
- отображение сторис после публикации.

---

### 2. Приватность сторис
- настройка ограничения доступа к сторис;
- исключение пользователей из просмотра;
- проверка применения настроек приватности;
- проверка недоступности сторис для исключённых пользователей.

---

### 3. Отображение сторис
- корректность отображения сторис у других пользователей;
- проверка отображения сторис при разных настройках доступа.

---

### 4. Управление сторис
- наличие элементов управления;
- доступность функций редактирования;
- отображение меню сторис (кнопка «три точки»).

---

## Структура тест-кейсов

Каждый тест-кейс содержит:

- ID тест-кейса  
- Название  
- Предусловия  
- Шаги выполнения  
- Ожидаемый результат (ОР)  
- Фактический результат (ФР)  
- Окружение  
- Результат выполнения  

---

## Окружение

Тестирование проводилось на:

- iPhone 14 Pro Max  
- iOS 26.6.4  

---

## Результаты выполнения

В ходе выполнения тест-кейсов были получены следующие результаты:

- Passed — сценарий отработал корректно  
- Failed — обнаружен дефект  

Обнаруженные проблемы:

- некорректная работа ограничений доступа к сторис;
- исключённые пользователи видят сторис;
- отсутствие элементов управления сторис;
- некорректная логика приватности.

---

## Примечание

Тест-кейсы оформлены в табличном виде и отражают реальные сценарии использования приложения, включая взаимодействие между несколькими пользователями.
