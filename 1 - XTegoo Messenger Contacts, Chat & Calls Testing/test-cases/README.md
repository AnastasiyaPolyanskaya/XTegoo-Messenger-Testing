# Test Cases - XTegoo Messenger (Contacts, Chat & Calls)

В данном разделе представлены тест-кейсы для проверки функциональности мобильного приложения XTegoo Messenger.

Тест-кейсы покрывают основные пользовательские сценарии, связанные с работой контактов, чатов и звонков.

---

## Цель тестирования

Обеспечить проверку корректной работы ключевых функций приложения и выявить дефекты в логике и отображении.

---

## Область покрытия

Тест-кейсы охватывают следующие функциональные области:

### 1. Контакты
- предоставление доступа к контактам через настройки устройства;
- работа с частичным доступом к контактам;
- работа с полным доступом;
- отображение контактов в приложении.

---

### 2. Чаты
- отображение контактов в списке чатов;
- корректность отображения времени сообщений;
- переходы между экранами;
- взаимодействие с элементами интерфейса.

---

### 3. Звонки
- совершение исходящих звонков;
- получение входящих звонков;
- отображение истории звонков;
- обновление имени контакта;
- удаление звонков.

---

## Структура тест-кейсов

Каждый тест-кейс содержит:

- ID тест-кейса  
- Название  
- Предусловия  
- Шаги выполнения  
- Ожидаемый результат (ОР)  
- Фактический результат (ФР)  
- Окружение  
- Результат выполнения  

---

## Окружение

Тестирование проводилось на следующем устройстве:

- iPhone 14 Pro Max  
- iOS 26.6.4 / 26.6.5  

---

## Результаты выполнения

В ходе выполнения тест-кейсов были получены следующие результаты:

- Passed — сценарий отработал корректно  
- Failed — обнаружен дефект  

Часть тест-кейсов завершилась с ошибками, что позволило выявить проблемы в работе:

- отображения контактов при ограниченном доступе;
- входящих звонков;
- удаления истории звонков;
- синхронизации времени в чате.

---

## Примечание

Тест-кейсы оформлены в табличном виде и отражают реальные пользовательские сценарии взаимодействия с приложением.
