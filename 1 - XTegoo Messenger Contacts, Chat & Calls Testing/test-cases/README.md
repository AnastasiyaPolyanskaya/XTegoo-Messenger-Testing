# Test Cases - XTegoo Messenger (Contacts, Chat & Calls)

В данном разделе представлены тест-кейсы для проверки функциональности мобильного приложения XTegoo Messenger.

Тест-кейсы охватывают ключевые сценарии работы с контактами, чатами, звонками и управлением сообщениями.

---

## Цель тестирования

Проверить корректность работы основных функций приложения и выявить дефекты в логике и интерфейсе.

---

## Область покрытия

### 1. Контакты
- доступ к контактам;
- отображение контактов.

---

### 2. Чаты
- отображение чатов;
- отправка сообщений;
- отображение времени сообщений.

---

### 3. Управление сообщениями

- проверка наличия кнопки «Удалить у себя»;
- проверка наличия кнопки «Удалить у всех»;
- корректность отображения вариантов удаления;
- понятность интерфейса удаления сообщений;
- соответствие поведения ожиданиям пользователя.

---

### 4. Звонки
- входящие звонки;
- исходящие звонки;
- история звонков;
- удаление звонков.

---

## Структура тест-кейсов

Каждый тест-кейс содержит:

- ID  
- Название  
- Предусловия  
- Шаги  
- Ожидаемый результат  
- Фактический результат  
- Окружение  
- Результат  

---

## Окружение

- iPhone 14 Pro Max  
- iOS 26.6.4  

---

## Результаты

В ходе выполнения тест-кейсов были выявлены:

- дефекты логики работы контактов;
- ошибки в звонках;
- проблемы отображения;
- отсутствие явной логики удаления сообщений;
- некорректный UX при удалении сообщений.
